Contribute

RailsCamp Germany is a non-profit event "for the community, from the community", therefore it is essential that as many people as possible help us to make this a great event for everybody. Any little help is appreciated, no contribution is too small.

Help with event organization

As a decentralized organization team, we’ve set up some tools to help us collaborate remotely. If you want to get get involved before the event, check out what we already have, find something you’d like to help with or add something new, and join the conversation:

  • GitHub Issues on the event repo are our main organization channel. Just pick an issue that you want to help with, create issues that are missing, or provide helpful information on existing issues.
  • #railscamp_de on Freenode is our offical IRC channel. Just drop by anytime.
  • The orga mailing list is the only non-public channel, which we only use for confidential information. If you want access, just request it. It’s not necessary for anything other than looking at a few boring numbers, though. ;)

We’re also doing regular Google Hangouts every Tuesday at 5pm. The URL is always shared on both IRC and Twitter.

Organize a session

The Railscamp is organized like a Barcamp, so the content of the event depends on you, the participants. You can help to make it awesome by planning a great session and pitching it to the other participants. Session can be related to Rails & Development, but also anything(!) else that you are excited about. Example sessions could be things like:

  • "My awesome new gem"
  • "This cool new feature in Rails 4
  • "You are doing it wrong!"
  • "Anonymous Coffee Nerd Support Group"
  • "Bike repairing 101"
  • ...

So, get going and start thinking about your own session!
